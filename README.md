This project is a personal landing page for **Sergey Kuznetsov**, focused on positioning as an enterprise-level system architect.

The landing emphasizes:

- Event-driven systems

- Audit & data integrity
- AI governance / RAG systems

This is not a portfolio or template site.

The goal is to communicate:

- understanding of scaling challenges
- experience with production-level architecture
